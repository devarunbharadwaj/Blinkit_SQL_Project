Blinkit SQL Project

This project is an end-to-end SQL-based analysis of Blinkit’s business data.
The goal of the project is to practice SQL querying, analyze real business metrics, and support insights using visual charts.
Project structure:-
  Blinkit_SQL_Project/
  │
  ├── sql_queries/              # All SQL scripts for Delivery, Marketing, Customer, Orders, Feedback
  ├── visuals/                  # PNG charts generated from the SQL outputs
  ├── data_dictionary/          # (Not used yet – kept for future addition)
  ├── insights_summary/         # (Not used yet – kept for future addition)
  └── README.md

What This Project Covers
1. Delivery Performance Analysis
Monthly delivery trends
Avg delivery time by partner
Delay rate
Delivery time vs distance

2. Marketing Analysis
Monthly spend vs revenue
CTR, CVR, ROAS by channel
Top campaigns by ROAS
Revenue by target segment

3. Customer Analysis
Segments
AOV per segment
Monthly new customers
Top locations
Revenue potential

4. Customer Feedback
Rating distribution
Monthly rating trend
Sentiment breakdown
Top negative issues
Rating by customer segment
Monthly feedback volume

5. Orders Analysis
Order distribution
Repeat customer behavior
Revenue patterns
